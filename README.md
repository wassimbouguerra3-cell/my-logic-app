# Key Features
- **Interactive Quiz Engine:** 25 curated puzzles focusing on formal logic, paradoxes, ethics, and consciousness.
- **Dynamic User Interface:** Sleek, modern design with a focus on immersive aesthetics.
- **Localization:** Seamless language switching between English (EN) and Arabic (AR).
- **Audio Experience:** Integrated music toggle for an enhanced, atmospheric gaming session.
- **Visual Effects:** Implemented a unique mouse-tracking light effect to guide user interaction.
- **Psychological Profiling:** Concludes with an MBTI-style personality assessment based on user responses.

# Tech Stack
- **Framework:** Next.js / React (for high-performance routing and state management).
- **Styling:** CSS3 (Custom animations, glow effects, and typography).
- **Deployment:** Vercel (Optimized for fast global delivery).

# Technical Evolution
This project represents a fusion of my previous learning stages:
- **UI/UX Maturity:** Building upon the responsive principles of my e-commerce project, I implemented more complex UI states (music, language, cursor tracking).
- **Logic & State:** Transitioned from simple quiz logic to a complex, multi-layered question engine that calculates personality traits.
- **Deployment Strategy:** Leveraging Vercel for a seamless, professional hosting environment.

# How to Play
1. Navigate to the Live Demo.
2. Choose your language (EN/AR) at the top.
3. Toggle the music to set the mood.
4. Answer the paradoxes and puzzles carefully.
5. Discover your personality profile at the end.
